# Edmodo Academy (EDA) Website

这是一个为 Edmodo Academy (EDA) 设计的现代化网站实现，包含以下页面：

- **搜索列表页** - 课程搜索和筛选功能
- **课程详情页** - 课程详细信息展示
- **登录页** - 用户认证界面

## 技术栈
- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (ES6+)
- 响应式设计
- 移动优先
